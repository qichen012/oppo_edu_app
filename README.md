# oppo_edu_app
第十九届软件创新大赛

## 项目简介
Oppo Edu App 是一个基于 AI 的智能教育辅助系统，为学习者提供全方位的知识管理和个性化学习支持。

### 技术架构与工作流程
系统采用 **FastAPI** 构建高性能 RESTful API 服务，当用户上传 PDF 学习资料时，**PyMuPDF** 快速提取文本内容，随后通过 **智增增 LLM** 进行智能解析，自动生成结构化的知识卡片（包含标题、摘要、关键词等），这些卡片通过 **Mem0** 长期记忆系统持久化存储，构建用户的个人知识图谱。在学习过程中，用户可以通过截图功能捕获屏幕内容，系统将图像编码为 base64 格式后再次调用 **智增增 API** 进行多模态分析，智能识别截图中的知识点并自动关联到已存储的笔记卡片中。

同时，系统集成 **DeepSeek** 大语言模型作为智能对话引擎，结合 Mem0 中存储的用户记忆和上下文，提供个性化的学习问答服务，所有对话历史通过 **JSON 文件系统** 分类存储。为了提供精准的内容推荐，系统使用 **Pandas** 和 **NumPy** 构建基于 Sigmoid 函数的用户行为评分模型，分析用户的点击和停留时长，然后通过 **DuckDuckGo Search** 和 **Feedparser** 从互联网和 RSS 订阅源中获取相关学习资源，实现智能推荐。此外，系统还支持会议音频转录功能，通过外部转录服务将音频文件转换为文本，便于后续的知识整理和回顾。

整个系统采用模块化设计，各技术组件无缝协作，从内容输入、智能处理、知识存储到个性化推荐，形成完整的智能学习闭环。

## 安装

1. 克隆项目：

```bash
git clone git@github.com:qichen012/oppo_edu_app.git
cd oppo_edu_app
```

2. 安装依赖：

```bash
pip install -r requirements.txt
```

## 使用方法

1. 启动服务：

```bash
python run/server.py
```

2. 运行测试：

```bash
pytest test/
```

## 后端技术栈

Oppo Edu App 的后端采用了现代化的技术栈，专注于智能教育和知识管理：

### 核心框架
- **FastAPI**：现代化高性能 Web 框架，提供异步支持和自动 API 文档生成
- **Pydantic**：数据验证和类型提示，确保 API 输入输出的类型安全

### AI 与大语言模型
- **OpenAI API**：集成多个 LLM 服务提供商
  - **DeepSeek**：用于智能对话和上下文理解
  - **智增增 API**：用于截图分析和内容理解
- **自定义 Prompt Engineering**：优化的提示词工程，提升 AI 输出质量

### 知识管理与记忆系统
- **Mem0**：长期记忆管理系统，实现用户知识的持久化存储和检索
- **向量存储**：支持语义搜索和智能推荐

### 文档处理
- **PyMuPDF (fitz)**：高效的 PDF 文本提取和处理
- **文本解析**：自动化提取、结构化存储学习笔记

### 数据处理与推荐
- **Pandas & NumPy**：数据分析和数值计算
- **自定义推荐算法**：基于 Sigmoid 函数的用户行为评分系统
- **DuckDuckGo Search (ddgs)**：智能内容搜索
- **Feedparser**：RSS 订阅源解析和内容聚合

### 多模态分析
- **图像分析**：截图智能识别和内容提取
- **音频转录**：会议音频自动转录为文本（集成外部转录服务）

### 数据存储
- **JSON 文件系统**：轻量级数据持久化，存储用户对话历史、知识卡片和记忆数据
- **结构化存储**：分类管理聊天历史、处理笔记、用户记忆等

### API 特性
- **RESTful API**：标准化的 HTTP 接口设计
- **自动文档生成**：FastAPI 内置的 Swagger/OpenAPI 文档
- **异步处理**：支持高并发请求和长时间任务

### 开发工具
- **Git**：版本控制
- **Python 虚拟环境**：依赖隔离和环境管理

通过以上技术栈，Oppo Edu App 构建了一个智能化的教育辅助系统，能够处理 PDF 文档、图像截图、音频转录，并结合大语言模型提供个性化的学习推荐和智能对话服务。
